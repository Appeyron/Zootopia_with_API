# Zootopia Animal Website Generator

This project generates a simple website with information about animals.

The user enters the name of an animal, and the program fetches data from the API Ninjas Animals API. The information is then displayed in a generated HTML website. If the animal is not found, the website will show an appropriate message.

## Installation

Clone the repository and install the required dependency:

```bash
pip install requests
```

## Usage

Run the program:

```bash
python main.py
```

Example:

```text
Enter a name of an animal: Fox
Website was successfully generated to the file animals.html.
```

After running the program, open `animals.html` in your browser to view the generated website.
